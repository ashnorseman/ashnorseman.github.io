  • CSS3 文字

    文字

    自动添加连字符

    再也不需要text-align: justify了。

    p {
    hyphens: auto;
}

    强行换行

    使用 Unicode 提供的换行符：0x000A。

    dd::after {
    content: "\A";
    
    // 防止和其他空白字符互相吞噬
    white-space: pre;

  • CSS3 视觉效果

    阴影

    单边阴影

    重点：利用 box-shadow 的负值扩展。

    div {
    box-shadow: 0 5px 4px -4px black;
}

    相邻两侧阴影

    收缩阴影的同时，加以位移。

    div {
    box-shadow: 3px 3px 6px -3px black;

  • CSS3 制作各种形状

    椭圆

    自适应椭圆

    用百分比定义 border-radius 的值。

    div {
    border-radius: 50%;
}

    半个椭圆

    分别定义：

    • border-top-left-radius
    • border-top-right-radius
    • border-bottom-right-radius
    • border-bottom-left-radius

    或用 border-radius 的缩写：左上 右上 右下 左下。

  • CSS3 背景与边框

    边框

    半透明边框

    • 使用 rgba() 或 hsla() 颜色
    • 但是……会有问题……
    div {
    // 白色背景和白色半透明边框重叠在一起！
    border: 10px solid hsla(0, 0%, 100%, .5); 
    background: white;
}

    解决方案：修改background-clip属性。

    • border-box（默认）
    • padding-box（留出边框）
    div {
    background-clip: padding-box;
}
